# 《忧黎》设计文档

## 基本信息

- 游戏名称：《忧黎》
- 宣传语：那盏补光灯熄灭之前，你是我唯一想带走的战利品。
- 类型：移动端网页纯文字 Galgame
- 结构：短篇，单线程多结局
- 攻略角色：忧黎

## 风格定位

70% 温柔治愈 + 30% 细碎心痛。

故事核心不是“拯救”，而是两个不擅长被爱的人，学习如何靠近彼此。

玩家扮演第一人称的“你”，不露脸，不预设性别。

## 技术方案

使用纯前端：

- HTML
- CSS
- JavaScript

无需后端，适合 GitHub Pages 部署。

## 当前系统

### 剧情节点

剧情存放于：

```text
data/story.js
```

每个节点格式：

```js
node_id: {
  chapter: "第一章",
  title: "匹配",
  speaker: "忧黎",
  text: "“下把一起？”",
  next: "next_node_id"
}
```

### 选项节点

```js
choices: [
  {
    label: "【报点】选项文本",
    next: "next_node_id",
    effects: { trust: 1, honesty: 1 }
  }
]
```

### 隐藏变量

- `trust`：信任
- `distance`：距离感
- `pressure`：压力
- `honesty`：坦诚
- `support`：稳定陪伴

第一版 Demo 暂不进行正式结局判定，但系统已保留变量。

## 抑郁设定处理原则

- 不将疾病作为萌点
- 不让爱情替代治疗
- 不将玩家塑造成唯一拯救者
- 允许忧黎有专业性、幽默、主动性和脆弱
- 复诊与服药应被写成正常生活的一部分
